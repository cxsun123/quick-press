## 依赖关系

```
Phase 1 (DB + 类型) ──┬── Phase 2 (侧边栏) ──┬── Phase 3 (编辑器布局)
                      │                       │
                      ├── Phase 4 (AI 摘要) ──┘
                      │
                      └── Phase 5 (可见度) ── Phase 3 ── Phase 7 (测试)
                                                    │
                      Phase 6 (MCP) ────────────────┘
```

- Phase 1 是所有后续 Phase 的硬依赖（必须先完成）
- Phase 2 / 4 / 5 / 6 可并行开发
- Phase 3 依赖 Phase 2（需要可折叠侧边栏）和 Phase 4（需要 summary 字段）
- Phase 7 是最终集成验证，需所有功能完成
